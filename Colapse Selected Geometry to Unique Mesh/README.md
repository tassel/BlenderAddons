# Collapse Selected Geometry to Unique Mesh - BlenderAddons

## <code>**COLLAPSE GEO 2 UNIQUE MESH**</code>
>[!NOTE]
>* Checks if the selected object uses Geometry Nodes, and if so, inserts a Realize Instances node (if not already present).<br>
>* Applies the Geometry Nodes modifier.<br>
>* Makes the object real (if it's an instance).<br>
>* Ensures it's a unique mesh (not linked to others).<br>
>
> Tested with Blender 4.2.2 / 4.2.3 (LTS)
